## Self-Hosting

### Prerequisites

- [Bun](https://bun.sh/) (v1.1+)
- A [Convex](https://convex.dev/) account (free tier works)
- At least one AI provider API key (Anthropic, OpenAI, or Google)

### Setup

1. **Clone and install dependencies**

```bash
git clone https://github.com/ourin-ai/ourin.git
cd ourin
bun install
```

2. **Set up Convex**

```bash
cd web
npx convex dev
```

This will prompt you to log in and create a new Convex project. Keep this terminal running.

3. **Configure environment variables**

Create `web/.env.local`:

```bash
# Convex (auto-generated by `npx convex dev`)
CONVEX_DEPLOYMENT=your-deployment-name
NEXT_PUBLIC_CONVEX_URL=https://your-deployment.convex.cloud

# AI Providers (add at least one)
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_API_KEY=...

# Security (generate with: openssl rand -base64 32)
CHAT_AUTH_SECRET=your-random-secret-min-32-chars

# Client-side config
NEXT_PUBLIC_SELF_HOSTING=true
```

Set the same `CHAT_AUTH_SECRET` in Convex:

```bash
npx convex env set CHAT_AUTH_SECRET your-random-secret-min-32-chars
```

4. **Start the app**

In a new terminal:

```bash
cd web
bun run dev
```

Visit `http://localhost:3000`
